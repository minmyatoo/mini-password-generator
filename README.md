## Description

This repository contains a versatile password generator that can be used through a web-based interface and the command line. Customize password length and character sets according to your preferences.

## Web Interface Usage

1. Clone the repository.
2. Open `index.html` in your web browser.
3. Configure your password preferences.
4. Click "Generate Password."
5. Your new password will be displayed!

## Command Line Usage

1. Clone the repository.
2. Open a terminal in the repository directory.
3. Run the following command:
   
```javascript
node password-generator.js [options]
```

   Replace `[options]` with the desired command-line options (e.g., `--length 12 --capital --numbers`).
