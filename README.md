# 🔐 Random Password Generator

Generate strong and secure random passwords both through a web interface and the command line! 💪

## Description

This repository contains a versatile password generator that can be used through a web-based interface and the command line. Customize password length and character sets according to your preferences.

## Web Interface Usage

1. Clone the repository.
2. Open `index.html` in your web browser.
3. Configure your password preferences.
4. Click "Generate Password."
5. Your new password will be displayed!

## Command Line Usage

1. Clone the repository.
2. Open a terminal in the repository directory.
3. Run the following command:
   
```javascript
node password-generator.js [options]
```

   Replace `[options]` with the desired command-line options (e.g., `--length 12 --capital --numbers`).

## Features

✅ Customizable password options through the web interface.
📏 Specify password length.
🔡 Include/exclude capital and small letters.
🔢 Include/exclude numbers.
🔣 Include/exclude special characters.
🛡️ Ensure a minimum number of numbers and special characters.

🚀 Generate passwords on the command line with ease.

## Technologies Used

- HTML
- CSS (Bootstrap for styling)

- Node.js (for command-line functionality)
